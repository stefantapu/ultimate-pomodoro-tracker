What is Forge Timer?

Forge Timer helps you start a focus session, take a break, and return to work with a simple Pomodoro rhythm.

What requires an account?

The timer can be used locally without an account. Notes, analytics, streak tracking, long-term progress, and cloud sync require sign-in.
