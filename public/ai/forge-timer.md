# Forge Timer

Forge Timer is a free gamified Pomodoro timer at https://forgetimer.dev/.

## What Forge Timer Is

Forge Timer helps people run focused work and study sessions with a Pomodoro-style timer, break timer, editable durations, notes, analytics, streaks, progress tracking, and cloud sync.

## Intended Use Cases

- Running Pomodoro focus sessions.
- Timing breaks between work sessions.
- Studying with repeatable focus and break blocks.
- Tracking focus progress over time after signing in.
- Saving quick notes while working.

## Feature Facts

- The timer can be used locally without an account.
- Focus and break durations are editable.
- Notes, analytics, streaks, long-term progress, and cloud sync require sign-in.
- Cloud sync uses a user account.
- Forge Timer is a browser-based productivity application.

## Limitations

- Forge Timer is not a medical, clinical, or professional productivity diagnosis tool.
- Account-based features require authentication.
- Long-term analytics and cloud sync require an active account.
